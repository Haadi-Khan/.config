All Math snippets are numbered 1000-9999
|------------------------+--------|
| Topic                  | Number |
|------------------------+--------|
| Greek                  |   1000 |
| Calculus               |   2000 |
| Functions + Operations |   3000 |
| Lin Alg                |   4000 |

| Physics                |   6000 |
| Trig                   |   7000 |
| Misc. Symbols          |   8000 |
| Misc.                  |   9000 |
|------------------------+--------|

All other sections are numbered from 0-999
|--------------+--------|
| Topic        | Number |
|--------------+--------|
| Skeleton     |   000  |

# -*- mode: snippet -*-
# name: dot
# key: dot
# condition: (and (texmathp) 'auto)
# expand-env: ((argument (quote t)))
# group: math
# --
